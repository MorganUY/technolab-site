# technolab-site
 ТехноЛаб — Сервисный центр

Официальный сайт сервисного центра «ТехноЛаб» — современная веб-платформа для заказа ремонта техники, просмотра каталога услуг и товаров, а управления заказами.

## О проекте

ТехноЛаб — это PHP-сайт на базе MySQL, предоставляющий полный цикл услуг по ремонту компьютеров, телефонов, телевизоров и другой техники. Проект включает в себя каталог товаров и услуг, корзину покупок, личные кабинеты клиентов, систему заявок и админ-панель.

## Основные возможности

### Для клиентов
- Просмотр каталога товаров и услуг с фильтрацией по категориям
- Подробная информация о каждом товаре/услуге
- Добавление в корзину и оформление заказов
- Персональный кабинет с историей заказов
- Избранные товары
- Запрос обратного звонка
- Регистрация и авторизация

### Административные функции
- Управление товарами и категориями
- Управление заказами и статусами
- Просмотр заявок на звонки
- Управление пользователями
- Аналитика и отчетность

## Технологический стек

- **Backend**: PHP 8.2+
- **База данных**: MySQL 8.0
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Отображение данных**: XSLT (XML-каталог товаров)
- **Шаблоны**: PHP-шаблоны с разделением логики и представления
- **Стили**: CSS Grid/Flexbox, адаптивная верстка
- **Скрипты**: Нативный JavaScript (script.js)
