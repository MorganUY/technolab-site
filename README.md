# ТехноЛаб — Сервисный центр

Официальный сайт сервисного центра «ТехноЛаб» — веб-приложение для просмотра услуг и товаров, а также оформления заказов.

## О проекте

Проект представляет собой сайт на PHP с элементами интернет-магазина и сервисного центра.
Реализованы каталог товаров и услуг, система заказов и пользовательский интерфейс.

В рамках лабораторной работы дополнительно реализована работа с XML.

## Основные возможности

### Для пользователей

* Просмотр каталога товаров и услуг
* Фильтрация по категориям
* Просмотр подробной информации
* Добавление в корзину
* Регистрация и авторизация

### Для администратора

* Управление товарами
* Управление заказами
* Работа с пользователями

## Технологии

* **Backend**: PHP
* **База данных**: MySQL
* **Frontend**: HTML, CSS, JavaScript
* **XML**: хранение данных о товарах
* **XSLT**: преобразование XML в HTML
* **XSD**: валидация структуры XML

## Работа с XML

В проекте используется файл `products.xml`, который содержит данные о товарах.

Реализовано:

* чтение XML с помощью PHP (`simplexml_load_file`)
* отображение данных на сайте
* проверка XML по схеме (`products.xsd`)
* преобразование XML в HTML с помощью XSL (`products.xsl`)

XML в данном проекте используется как источник данных, аналог базы данных.

## Структура проекта

* `products.xml` — данные о товарах
* `products.xsd` — схема проверки
* `products.xsl` — преобразование в HTML
* `index.php` — главная страница
* `catalog.php` — каталог товаров
* `product.php` — карточка товара
